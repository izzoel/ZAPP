# ZAPP

**ZAPP** adalah **template framework berbasis Laravel** yang dirancang untuk mempercepat pengembangan aplikasi web modern dengan arsitektur rapi, reusable, dan siap produksi.

ZAPP menggabungkan:
- **Laravel** sebagai core backend framework
- **Livewire** untuk interaksi dinamis tanpa kompleksitas SPA
- **Bootstrap** sebagai UI framework yang stabil dan familiar

Template ini cocok untuk aplikasi **admin panel**, **sistem informasi**, **dashboard internal**, dan **aplikasi enterprise**.

---

## Fitur Utama

- Laravel versi terbaru
- Livewire (server-driven UI)
- Bootstrap 5
- Struktur folder rapi dan konsisten
- Siap dikembangkan untuk role & permission
- Template UI backend siap pakai
- Modular dan scalable
- Mudah dikustomisasi untuk berbagai kebutuhan proyek

---

## Teknologi yang Digunakan

- **Laravel** – Backend framework
- **Livewire** – Dynamic UI tanpa JavaScript berat
- **Bootstrap 5** – CSS framework
- **Blade Template Engine**
- **MySQL / MariaDB** (default)
- **Vite** (asset bundling)
